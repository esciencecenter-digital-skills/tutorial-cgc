# Data

The data is a low-resolution version of the spring-index data set from [Emma Izquierdo-Verdiguier, Ra√∫l Zurita-Milla and Toby R. Ault, Mark D. Schwartz, Agricultural and Forest Meteorology 262, 34-41 (2018)](https://doi.org/10.1016/j.agrformet.2018.06.028) (if you use the data, please cite the paper). 
The spatial dimensions have been down-sampled by a factor 25. The average spring-index values computed for each (25x25) grid-cell windows have been rounded off to the closest integer.
